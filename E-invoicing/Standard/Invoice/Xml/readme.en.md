# XML e-Invoicing Standard

## Overview

The **Generix Spain XML e-Invoicing Standard** provides a unified and modular XSD schema for the representation of electronic invoices in XML format.

It has been designed to facilitate B2B integrations, ensure interoperability between systems and support compliance with Spanish electronic invoicing regulations.

## Key Features

### Compatibility

Complies with Spanish electronic invoicing regulations, including the requirements derived from the **Crea y Crece Law** and applicable tax regulations.

### Modularity

The solution is based on reusable complex types that allow different document structures to be composed from common building blocks.

### Validation

The XSD schemas provide structural and datatype validation for every XML document exchanged.

### Scalability

Designed to support high transaction volumes while maintaining integrity, consistency and performance.
