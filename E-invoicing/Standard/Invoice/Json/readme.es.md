# Estándar JSON e-Invoicing

## Descripción General

El estándar **JSON e-Invoicing de Generix Spain** proporciona una estructura unificada y modular para la representación de facturas electrónicas en formato JSON.

Este estándar ha sido diseñado para facilitar la integración con sistemas B2B, garantizando la interoperabilidad entre plataformas y el cumplimiento de los requisitos normativos aplicables a la facturación electrónica en España.

## Características Principales

### Compatibilidad

Cumple con los requisitos establecidos por la normativa española de facturación electrónica, incluida la **Ley 18/2022 (Crea y Crece)** y los requisitos fiscales aplicables.

### Modularidad

La estructura está organizada de forma flexible para permitir la inclusión únicamente de los elementos necesarios en función del tipo de factura y del escenario de negocio.

### Validación

Incluye esquemas y reglas de validación que permiten verificar la integridad, consistencia y calidad de los datos intercambiados.

### Escalabilidad

Ha sido diseñado para soportar grandes volúmenes de transacciones sin comprometer el rendimiento ni la capacidad de procesamiento.
